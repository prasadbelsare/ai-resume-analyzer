# AI Resume Analyzer

A React-based AI resume analyzer that allows users to upload resumes and receive AI-driven feedback such as ATS scores and job matching insights.

## Tech Stack

- React  
- TypeScript  
- Vite  
- Tailwind CSS  
- Puter.js  
- Zustand  

## Features

- Client-side authentication
- Resume upload and secure storage
- AI-based resume evaluation
- ATS scoring and feedback
- Responsive and reusable UI components

## Setup

```bash
https://github.com/prasadbelsare/ai-resume-analyzer.git
cd ai-resume-analyzer
npm install
npm run dev
```
Open http://localhost:5173 in your browser.
